## Run e Build

Usamos ViteJS como build tool.

Para rodar:

```
yarn && yarn dev && yarn css || yarn && yarn dev --host && yarn css
```

Para buildar:

```
yarn build
```

Para rodar um preview:

```
yarn preview
```
